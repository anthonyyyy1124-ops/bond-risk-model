# Bond Risk Assessment Model

This project develops a simplified interest rate risk assessment framework for corporate bond underwriting.

## Features
- Bond pricing
- Duration & DV01 calculation
- Yield curve stress testing
- Value at Risk (VaR)

## Files
- model.py: main Python model
- Bond_Data.xlsx: input data
- Risk_Output.xlsx: sample output

## Usage
Run the Python script and input bond data via Excel.

## Purpose
This tool is designed as a transparent and interpretable reference model for interest rate risk analysis.
